# CLI weather

Простое консольное приложение на Node.JS для вывода погоды

Установка:

```shell
npm i -g cli-weather-app
```

[Получить токен](https://home.openweathermap.org/api_keys)

Добавить токен:
```shell
weather -t <[YOU TOKEN]>
```

Добавить город:
```shell
weather -с <[CITY NAME]>
```

Запуск приложения:
```shell
weather
```
